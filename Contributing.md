# Contributing to Payments & Ledger Engine

Thanks for your interest in contributing.

## Ways to Contribute
- Report bugs via GitHub Issues
- Suggest improvements to architecture or documentation
- Submit pull requests for bug fixes or enhancements

## Before Submitting a PR
- Make sure all existing tests pass: `./mvnw test`
- Add tests for any new behaviour
- Follow the existing code style and package structure
- Reference the relevant architecture doc in your PR description
